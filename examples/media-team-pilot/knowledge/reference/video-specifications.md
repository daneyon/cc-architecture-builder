# Video Specifications

## Target Output

| Property | Value | Notes |
|----------|-------|-------|
| Resolution | 1920x1080 | Standard YouTube HD |
| Aspect ratio | 16:9 | Landscape format |
| Framerate | 30 fps | Standard for slideshow |
| Video codec | H.264 (libx264) | Universal compatibility |
| Audio codec | AAC | Standard for .mp4 |
| Container | .mp4 | YouTube recommended |
| CRF | 23 | Good quality/size balance |
| Audio bitrate | 192 kbps | Good quality narration |
| Audio sample rate | 44100 Hz | CD quality |

## YouTube Upload Requirements

| Property | Requirement |
|----------|-------------|
| Min resolution | 720p (1280x720) |
| Max file size | 256 GB |
| Max duration | 12 hours |
| Supported formats | .mp4, .mov, .avi, .wmv |
| Recommended codec | H.264 + AAC in .mp4 |
